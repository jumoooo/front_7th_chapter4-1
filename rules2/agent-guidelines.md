# Agent 작업 가이드라인 (Agent Guidelines)

## 📋 개요

Cursor Auto Agent가 작업을 수행할 때의 상세 가이드라인입니다.

---

## 🎯 핵심 원칙

1. **안정성**: 기능이 깨지지 않아야 함
2. **속도**: 빠르게 작업
3. **정확성**: 의도대로 정확한 작업
4. **최소한의 작업**: 불필요한 작업 최소화

---

## 📋 작업 프로세스

### 작업 시작 전

1. `WORK/current-step.md` 읽기
2. `WORK/next-step.md` 읽기
3. `WORK/core-principles.md` 읽기
4. Git 상태 확인

### 작업 중

1. 작은 단위로 작업
2. 즉시 검증 (`tsc --noEmit`)
3. 오류 발생 시 즉시 수정

### 작업 완료 후

1. 상태 파일 업데이트
2. Git 커밋 (선택적)

---

## ✅ 체크리스트

- [ ] 핵심 원칙 준수
- [ ] 타입 체크 통과
- [ ] 기능 회귀 없음
- [ ] 상태 파일 업데이트

---

**이 가이드라인을 따르면 안전하고 효율적으로 작업할 수 있습니다! 🚀**
